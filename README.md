# Food Insecurity Analysis Along the U.S.–Mexico Border

## Project Overview
This project integrates five major U.S. government data sources to create a comprehensive dataset on food insecurity, poverty, and food access across U.S. counties, with focus on the U.S.–Mexico border region.

## Data Sources
- **U.S. Census Bureau**: Unemployment, Poverty, Population, Education
- **USDA Food Access Research Atlas**: Food desert indicators, low-access populations
- **Feeding America Map the Meal Gap**: Food insecurity rates (2019-2023)

## Files
- `Deliverable 2- Data Inventory .R` - Main data cleaning and preparation script
- `Border_Food_Security_All_Data_Final.csv` - Complete merged dataset
- `Border_Food_Security_TX.csv` - Texas data
- `Border_Food_Security_NM.csv` - New Mexico data
- `Border_Food_Security_AZ.csv` - Arizona data
- `Border_Food_Security_CA.csv` - California data
- `Border_Counties_Only.csv` - Border counties only

## Border Counties Definition
21 U.S. counties that directly border Mexico across Texas (13), New Mexico (3), Arizona (4), and California (1).
